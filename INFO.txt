1. Sobre o Projeto
-----------------------------------------------------
Este projeto é uma aplicação web que utiliza a API oficial do IBGE 
para listar todos os estados do Brasil e, ao escolher um estado, 
mostrar todas as cidades pertencentes a ele.

A aplicação é composta por duas páginas principais:
- index.html → Exibe os estados
- cidade.html → Lista as cidades do estado selecionado

A comunicação com a API é feita usando AJAX via jQuery.

3. Como Executar o Projeto
-----------------------------------------------------
A aplicação está disponível online através do GitHub Pages, então
não é necessário rodar o projeto em servidor local.

Para visualizar e testar o sistema, basta acessar o link abaixo:
    https://gustavooxp.github.io/projeto-cidades-estados/

-----------------------------------------------------
4. Como Navegar
-----------------------------------------------------
1. Acesse o link do GitHub Pages.
2. A página inicial (index.html) carregará automaticamente a lista de estados usando a API do IBGE.
3. Após escolher um estado, o botão “Ver Cidades” será habilitado.
4. Ao clicar, você será enviado para cidade.html com o estado selecionado.
5. A próxima página exibirá todas as cidades ordenadas de A–Z.
6. Use o botão “Voltar” para retornar à página inicial.
